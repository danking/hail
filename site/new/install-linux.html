Hail

Install on Linux

  1. Install Java 8.
  2. Install a recent version of the C and C++ standard libraries. GCC 5.0, LLVM version 3.4, or any later versions include sufficiently recent versions.
  3. Install Python 3.6 or 3.7 and pip.
  4. Install LZ4.
  5. Install BLAS and LAPACK.
  6. Install hail using pip.

On a Debian-like system, the following should suffice: 

apt-get install openjdk-8-jre-headless \
                g++ \
                python3 python3-pip \
                liblz4 \
                libopenblas liblapack3
pip install hail
