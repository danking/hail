Hail has three cheatsheets. The first two discuss how to use the main two data structures in Hail: the Table and the MatrixTable. The third cheatsheet describes how to use Hail on the cloud.

Tables

Tables are the Hail data structure for one-dimensional data. You can create a Table from TSVs, CSVs, sites VCFs, FAM files, and Pandas DataFrames.

MatrixTables

MatrixTables are the Hail data structure for two-dimensional data. You can create a MatrixTable from VCF, BGEN, and PLINK files.

Cloud Clusters

Learn all the tips and tricks for using Hail on cloud clusters like Google Dataproc.
