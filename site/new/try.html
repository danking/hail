pip install ipython

Start an IPython session by copy-pasting the below into your Terminal. 

ipython

Let's randomly generate a dataset according to the Blading-Nichols Model. The dataset has one-hundred variants and ten samples from three populations. 

import hail as hl
mt = hl.balding_nichols_model(n_populations=3,
                              n_samples=10,
                              n_variants=100)
mt.show()

The last line, mt.show(), displays the dataset in a tabublar form.
